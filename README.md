Simple board with some SMD components.

- Hi-Link 100-240 VAC to 5 VDC
- HT7333A 3.3V voltage regulator
- 4x SONGLE  SRD-05VDC-SL-C
and other minor parts like resistors, capacitors etc...

Not tested but should work!
